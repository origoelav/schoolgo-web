# OrigGO APK - Site Oficial

Este é o repositório do site oficial do OrigGO, uma plataforma inteligente para otimização de rotas de entrega.

## Tecnologias Utilizadas

- **Vite**: Build tool rápida e moderna.
- **TypeScript**: Tipagem estática para maior segurança.
- **React**: Biblioteca para construção de interfaces.
- **shadcn/ui**: Componentes de UI acessíveis e estilizáveis.
- **Tailwind CSS**: Estilização baseada em utilitários.
- **Supabase**: Backend-as-a-Service para autenticação e banco de dados.

## Como Iniciar o Projeto Localmente

Siga os passos abaixo para rodar o ambiente de desenvolvimento:

1. **Clone o repositório**
   ```sh
   git clone <URL_DO_REPOSITORIO>
   ```

2. **Instale as dependências**
   ```sh
   npm install
   ```

3. **Inicie o servidor de desenvolvimento**
   ```sh
   npm run dev
   ```

## Deploy

O projeto pode ser buildado para produção usando:
```sh
npm run build
```

Os arquivos gerados na pasta `dist` podem ser hospedados em qualquer provedor de estático (Netlify, Vercel, GitHub Pages, etc).
